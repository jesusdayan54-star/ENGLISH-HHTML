# EXTRA · Anexo 06 — Falsos Amigos (False Friends)

> 📋 Palabras que se parecen al español pero significan algo distinto. Son una fuente constante de errores, incluso en niveles avanzados.

## Los más comunes

| Palabra en inglés | Falso significado (parece) | Significado real |
|---|---|---|
| **actually** | actualmente | en realidad |
| **currently** | — | actualmente |
| **embarrassed** | embarazada | avergonzado/a |
| **pregnant** | — | embarazada |
| **realize** | realizar | darse cuenta |
| **carry out** | — | realizar |
| **sensible** | sensible | sensato/a |
| **sensitive** | — | sensible |
| **assist** | asistir (ir a algo) | ayudar |
| **attend** | — | asistir (a un evento) |
| **library** | librería | biblioteca |
| **bookstore** | — | librería |
| **success** | suceso | éxito |
| **event** | — | suceso |
| **exit** | éxito | salida |
| **large** | largo | grande |
| **long** | — | largo |
| **fabric** | fábrica | tela |
| **factory** | — | fábrica |
| **argument** | argumento (de una historia) | discusión/pelea |
| **plot** | — | argumento (de una historia) |
| **rope** | ropa | cuerda |
| **clothes** | — | ropa |
| **college** | colegio | universidad |
| **school** | — | colegio |
| **parents** | parientes | padres |
| **relatives** | — | parientes |

## Ejemplos en contexto

> ❌ *I realized a project.* → ✅ *I carried out a project.* (realicé un proyecto)
> ❌ *She is sensible about criticism.* → ✅ *She is sensitive about criticism.* (sensible a las críticas)
> ❌ *I need to go to the library to buy a book.* → ✅ *I need to go to the bookstore to buy a book.* (librería)

## Consejo

Cuando dudes de una palabra que "se parece mucho" al español, verifícala — es justo ese parecido lo que genera el error.
