# A2 · Vocabulario 06 — Tecnología y Deportes (Technology & Sports)

## Tecnología básica

| Inglés | Español |
|---|---|
| Computer | Computadora |
| Screen | Pantalla |
| Password | Contraseña |
| App / Application | Aplicación |
| Wi-Fi connection | Conexión Wi-Fi |
| To download | Descargar |
| To charge (a phone) | Cargar (un teléfono) |
| Battery | Batería |

## Deportes comunes

| Inglés | Español |
|---|---|
| Soccer / Football | Fútbol |
| Basketball | Baloncesto |
| Swimming | Natación |
| Running | Correr |
| Tennis | Tenis |
| Gym | Gimnasio |

## Verbos con deportes: play, do, go

| Estructura | Ejemplo |
|---|---|
| **play** + deportes con pelota/equipo | *play soccer, play tennis* |
| **do** + deportes individuales | *do yoga, do gymnastics* |
| **go** + deportes en "-ing" | *go swimming, go running* |

## Frases útiles

> *My phone battery is low.* (La batería de mi teléfono está baja.)
> *I go to the gym three times a week.* (Voy al gimnasio tres veces por semana.)
> *Do you play any sports?* (¿Practicas algún deporte?)

## Consejos de pronunciación

- **Gym** /dʒɪm/ — la "g" suena como "j" en inglés.

## Práctica

Describe tu relación con la tecnología y el deporte usando play/do/go correctamente.
