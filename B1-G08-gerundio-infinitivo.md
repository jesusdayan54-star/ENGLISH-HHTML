# B1 · Vocabulario 01 — Rasgos de Personalidad (Personality Traits)

## Rasgos positivos

| Inglés | Español |
|---|---|
| Outgoing | Extrovertido/a |
| Reliable | Confiable |
| Ambitious | Ambicioso/a |
| Easy-going | Relajado/a, de trato fácil |
| Generous | Generoso/a |
| Honest | Honesto/a |
| Patient | Paciente |
| Confident | Seguro/a de sí mismo |

## Rasgos negativos

| Inglés | Español |
|---|---|
| Stubborn | Terco/a |
| Selfish | Egoísta |
| Arrogant | Arrogante |
| Lazy | Perezoso/a |
| Impatient | Impaciente |
| Rude | Grosero/a |
| Shy | Tímido/a |

## Describiendo a alguien

> *She's very **outgoing** — she loves meeting new people.*
> *He can be a bit **stubborn** sometimes.*
> *My best friend is really **reliable**; I can always count on her.*

## Collocations útiles

| Expresión | Significado |
|---|---|
| a good sense of humor | buen sentido del humor |
| a strong personality | una personalidad fuerte |
| to get on well with someone | llevarse bien con alguien |

## Consejos de pronunciación

- **Ambitious** /æmˈbɪʃəs/.
- **Stubborn** /ˈstʌbərn/.

## Práctica

Describe la personalidad de tres personas que conoces, usando al menos 5 adjetivos de esta lista.
