# C1 · Vocabulario 04 — Conceptos Abstractos (Abstract Concepts)

Vocabulario para discutir ideas, filosofía y temas complejos — típico de debates y ensayos de opinión.

## Sustantivos abstractos

| Inglés | Español |
|---|---|
| Autonomy | Autonomía |
| Integrity | Integridad |
| Resilience | Resiliencia |
| Legacy | Legado |
| Paradox | Paradoja |
| Dilemma | Dilema |
| Consensus | Consenso |
| Ambiguity | Ambigüedad |

## Adjetivos para ideas

| Inglés | Español |
|---|---|
| Controversial | Controvertido |
| Subjective / Objective | Subjetivo / Objetivo |
| Unprecedented | Sin precedentes |
| Inevitable | Inevitable |
| Counterintuitive | Contraintuitivo |

## Expresiones para discutir ideas

| Expresión | Significado |
|---|---|
| to challenge a notion | cuestionar una noción/idea |
| to strike a balance | encontrar un equilibrio |
| a double-edged sword | un arma de doble filo |
| food for thought | algo en qué pensar |

## En contexto

> *This raises an interesting **dilemma**.*
> *Her resilience in the face of adversity was remarkable.*
> *Technology is often a **double-edged sword**.*

## Consejos de pronunciación

- **Resilience** /rɪˈzɪliəns/.
- **Counterintuitive** /ˌkaʊntərɪnˈtuɪtɪv/.

## Práctica

Escribe una oración compleja usando "paradox" o "dilemma" sobre un tema de actualidad.
